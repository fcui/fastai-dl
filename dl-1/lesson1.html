第一课 猫狗识别¶

windows 10下的运行环境配置¶

  1. 安装anaconda 5.2版本并进行更新
    conda update --all

  1. 安装pythorch 0.4版本, 需搭配CUDA 9.0版本

    conda install pytorch -c pytorch
pip3 install torchvision

  1. 在运行过程中可能会出现 import PIL.Image 的错误, 这里需要

    conda uninstall pillow
pip intall pillow

    如果pip安装不了, 就去这里下载, 然后本地安装.

  1. 在本地下载fastai

    git pull https://github.com/fastai/fastai.git
